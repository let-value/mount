# mount

Module for listing, mounting and unmounting media devices

Direct port of [mount.py](https://github.com/vallentin/mount.py)

## Install

package.json

```json
{
    "dependencies": {
        "mount": "git://github.com/let-value/mount.git"
    }
}
```
